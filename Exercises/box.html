By default, the width and height only set the content box

  • This is the default behaviour of HTML elements when box-sizing is set to content-box
  • Padding and border will be added to the width and height

1

Box-sizing is set to border-box

  • Padding and border will be included width and height

Some elements have a outer display type of block by default, and some have inner. When elements use are only using these display properties, they are said to be in normal flow.

A span is an inline element.

A div and a p tag are block elements.

Block

  • Block elements assume the width of their container by default
  • Block elements break onto a new line
  • Width and height properties can be set
  • Padding, margin and border pushes elements away from the box

Inline

  • Box doesn't break onto new lines
  • Width and height properties do not work
  • Vertical padding, margins and borders apply but don't push other inline boxes away
  • Horizontal padding, margins and borders apply and will push other inline boxes away
